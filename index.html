企業後台管理 UI 設計

為企業客製化的後台管理網頁，客戶需要能夠快速資料瀏覽、出勤打卡、加班申請等功能，這樣的需求很適合使用 RWD 製作，能夠滿足 user 能夠隨時透過行動裝置使用此功能。
考量到行動裝置的使用，在 RWD 設計上以簡潔的 button 元素為主，button 元素強調回饋感，以確保 user 在操作時的流暢度。

Centech 飲水機

參考歐美網站流行的大型視覺排版，首頁使用 fullpage 加強瀏覽時的流暢度，並且使用些許的 scroll down 動畫增添質感。
這是進入這個行業的第二個作品，在對 RWD 都還不夠熟悉的狀況下，又要做出理想的效果，對當時的我來說算是蠻大的挑戰。

中央牌電風扇

客戶是銷售與開發國產電風扇的慶奇公司，原本既有的網站已不敷使用，網站的製作重點在於行動裝置的購物車功能，所以在不大幅改動網站架構的前提下，製作新的網站。
算是很中規中矩的購物車網站。

品醇洋酒網站

這是進入這個行業第一個接手的案子，客戶對網站的要求不高，也沒有想要做成 RWD 網站的意願，因此依照該公司的品牌印象去製作網站視覺
考量到未來可能會再製作購物車功能，因此在配置上，盡量以嘗間的購物車網站為原型。

這是 Glenn 的個人網站，存放一些圖像設計、畫作與網頁作品。在風格上做了不同於目前流行的 Flat style，反而是做了充滿動態 SVG 與手繪感十足的視覺設計:P

這是第二次嘗試「全手刻」網站(還是有使用 jQuery :P)，距離上一次這麼做已經經過一年了，這個網站包含了這一年所有成果的總結。

再參考許許多多的範例後，嘗試自己去做簡易的 Progress, lazyload, pictureView, modalBox, parallax...等功能，對於 Javascript 程度是初出茅廬的我而言，是非常大的挑戰，在邊嘗試邊修正的過程中，對於 Javascript 也有更深一層的認識。
